# Bank Rest API

<!-- ABOUT THE PROJECT -->
## О проекте

Создание микросервиса, клиент для работы с внешним API и базу данных для хранения данных о счетах и транзакциях между ними, и также о валютных курсах. 
Spring Boot, Spring JPA, Postgresql, Postman, (Ноут не позволяет изучить Docker, крашится сразу так как не тянет виртуализацию)
